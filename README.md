# macos-tools

A collection of small, self-contained utilities for macOS power users.

## Utilities

| Directory | Description |
|-----------|-------------|
| [`migrate-project/`](migrate-project/) | Migrate projects from iCloud-managed `~/Documents` to local storage, updating session metadata for Claude Code, Co-work, and Codex |

## Structure

Each utility lives in its own directory with its own README, tests, and scripts. No shared dependencies between utilities.
